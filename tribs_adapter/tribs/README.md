# tRIBS Executables Module

This module contains binaries for the tRIBS model for use in the tRIBS adapter and web application.

## Executables

* tRIBS: The serial version of the tRIBS model.
* tRIBSpar: The parallel version of the tRIBS model.

## Version

The version of the tRIBS executables included in this module is: 

5.3.1

## Source Code

The source code for the tRIBS model can be found on GitHub at: https://github.com/tribshms/tRIBS

## Building

Instructions for building tRIBS can be found here: https://tribshms.readthedocs.io/en/latest/man/Model_Execution.html#cmake
